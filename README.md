# ⏱️ 5 Sekund - Multiplayer Web Game

Internetowa, wieloosobowa wersja popularnej gry imprezowej "5 Sekund". Gra pozwala na tworzenie prywatnych pokoi, wspólną zabawę ze znajomymi w czasie rzeczywistym i rywalizację na wirtualnej planszy.

## 📖 O projekcie

Celem projektu jest przeniesienie dynamiki gry "5 sekund" do przeglądarki. Gracz ma dokładnie 5 sekund na wymienienie 3 rzeczy z wylosowanej kategorii. Ponieważ to gra imprezowa, weryfikacja odpowiedzi odbywa się poprzez demokratyczne głosowanie pozostałych graczy (Zaliczone / Skucha).

## ✨ Główne funkcje (Planowane)

* **System Pokoii (Lobby):** Tworzenie prywatnych gier za pomocą unikalnego kodu.
* **Czas Rzeczywisty:** Synchronizacja stanu gry, odliczania i ruchów graczy (Socket.io).
* **Wirtualna Plansza:** Interaktywna plansza z pionkami przesuwanymi po akceptacji odpowiedzi.
* **System Głosowania:** Weryfikacja odpowiedzi przez pozostałych uczestników gry.
* **Baza Pytań:** Generowane przez AI.

## 🛠️ Technologie (Tech Stack)

Projekt jest budowany z użyciem następujących technologii:
* **Frontend:** HTML5, CSS3 (Grid/Flexbox), JavaScript (Vanilla / React.js)
* **Backend:** Node.js, Express.js
* **Komunikacja:** Socket.io (WebSockets)
* **Baza Danych:** GroqCloud
